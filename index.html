PCから流れる音（ZOOMや動画の相手の声）を録音・文字起こしするには、Windowsの「ステレオミキサー」機能を使用します。

  1. タスクバー右下の「スピーカー」アイコンを右クリックし、「サウンドの設定」を開く。
  2. 下部にある「サウンド コントロール パネル」（または「その他のサウンド設定」）をクリック。
  3. 開いたウィンドウの「録音」タブを選択。
  4. 空いている場所で右クリックし、「無効なデバイスの表示」にチェック。
  5. 表示された「ステレオミキサー」を右クリックして「有効」にする。
  6. 当アプリの INPUT SOURCE で「ステレオミキサー」を選択する。
※Macをご利用の場合はステレオミキサー機能がないため、別途「BlackHole」などの仮想オーディオソフトのインストールが必要です。
※スマホ等、対応していないデバイスもあります。
